jVi Visual Block Mode

Selecting a Rectangular Region

There are two techniques for selecting a block.
  • mouse
    Drag the mouse so the the corners of the selection mark the rectangular region of interest. Then enter Ctrl-V to begin visual block mode. After Ctrl-V use the arrow keys.
  • arrow keys
    Enter Ctrl-V and then use the arrow keys to move move the cursor. In block mode, cursor motion adjust a corner of the block. See the following example to see how the cursor is moved to other corners of the block. Note that any vi/vim motion commands may be used to move the cursor and extend the block.
